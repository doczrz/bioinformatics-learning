# Bilingual Textbook

Open the interactive reader when the learner asks to view the course. Answer questions using the course version, lesson ID, section, language, and selected excerpt supplied by the reader.

Treat local R and SSH/HPC execution as unavailable in version 1. Do not claim that code was executed.
